☕ Lottle Cafe UI/UX Website

A modern and responsive cafe website designed to deliver a clean user experience and visually appealing interface. This project focuses on frontend development, UI design, and real-world business use cases.

---

🔗 Live Demo

👉 https://project-lottle-cafe-ui-ux.vercel.app/

---

🚀 Features

- Responsive design for mobile, tablet, and desktop
- Clean and modern UI/UX
- Interactive sections (Home, Menu, About, Contact)
- Smooth navigation
- WhatsApp integration for easy customer contact
- Optimized layout for real-world cafe business
- Contact Form integrated using Formspree
- Google Map Embedded

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

📌 Purpose

This project was built to:

- Practice real-world frontend development
- Create a business-ready website
- Improve UI/UX design skills
- Showcase portfolio work for internships and freelancing
